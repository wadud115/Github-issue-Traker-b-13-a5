ans 1 :
var is function-scoped and can be re-declared and re-assigned. It is considered outdated and can cause issues.

let is block-scoped and can be re-assigned but cannot be re-declared in the same scope.

const is block-scoped and cannot be re-assigned or re-declared. It is used for fixed values.

ans 2:

The spread operator is used to expand or copy elements of an array or object.

ans 3:

map(): returns a new array after modifying each element

filter(): returns a new array based on a condition

forEach(): only loops through the array and does not return anything

An arrow function is a shorter way to write a function in JavaScript.
